Welcome to GuoDongBa Project

GuoDongBa project is a solution including gym booking for customers and gym order managing for business. Customers can order gym using guodongba anroid application, business can visit our manage site GuoDong, register an account, and then recorde gyms, manage order lists.

Project Details

In our android app, user can:

  1. Register and login, change profile image, update app.
  2. Choose the hot gyms according to the distance.
  3. Choose the gyms according to the gym types.
  4. Book position of gyms in few days.
  5. Manage orders which are unused, already used and cancelled.

In our GuoDong site, gym manager can:

  1. Register and login, change profile.
  2. Add all kinds of gyms, for example, pingpang hall, badminton hall.
  3. Manage orders.
